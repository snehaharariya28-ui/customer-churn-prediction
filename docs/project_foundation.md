# Project Foundation

## Project Title

Customer Churn Prediction for Utility Services

## Problem Statement

Customer churn is a major challenge for utility companies because acquiring a new customer is often more expensive than retaining an existing one. The objective of this project is to analyze customer behavior and build a machine learning model capable of identifying customers who are likely to leave the service.

## Project Objectives

* Analyze customer data to identify factors contributing to churn.
* Perform exploratory data analysis to uncover patterns and trends.
* Engineer relevant features for predictive modeling.
* Build and evaluate machine learning models for churn prediction.
* Generate business insights and recommendations.
* Develop an interactive Power BI dashboard for stakeholder reporting.

## Dataset Overview

The dataset contains customer information, pricing details, sales channel information, consumption patterns, and churn status.

Target Variable:

* churn

  * 0 = Non-Churn Customer
  * 1 = Churn Customer

The project will be considered successful if the final model can effectively identify churn customers and provide actionable business insights that support customer retention strategies.
